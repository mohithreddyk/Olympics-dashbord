About This Application

Welcome to the Olympics Medals Tracker! This application allows users to:

  • View a table of Olympic medal counts by country.
  • Click on a country name to see detailed medal information.
  • Create and log in to accounts for a personalized experience (future feature).

Our mission is to provide a user-friendly platform for tracking Olympic achievements and exploring medal statistics in an interactive way.

Features:

  • Dynamic routing for country-specific medal details.
  • Responsive design for both desktop and mobile devices.
  • Built using Python, Flask, HTML, CSS, and JavaScript.
